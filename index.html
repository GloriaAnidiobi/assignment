HTML TAG EXPLANATION

Use of HTML Tags:

An HTML tag is a snippet of code used to define element within a webpage.

The Use of HTML tag Heading:

h1: Used for most important heading on the the page.

h2: Used for subheading.

h3: Used for subsection under h2

h4: Used for subtopics

h5: For further subdivision
h6: The least important heading often rarely used
